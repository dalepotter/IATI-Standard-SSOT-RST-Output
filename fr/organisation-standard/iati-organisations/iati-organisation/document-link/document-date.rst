document-date
=============

``iati-organisations/iati-organisation/document-link/document-date``

This is the reference page for the XML element ``document-date``. 

.. index::
  single: document-date

Definition
~~~~~~~~~~


The date of publication of the document that is being linked to.


Rules
~~~~~








This element must occur no more than once (within each parent element).







Attributes
~~~~~~~~~~


.. _iati-organisations/iati-organisation/document-link/document-date/.iso-date:

@iso-date
  

  This attribute is required.



  This value must be of type xsd:date.
