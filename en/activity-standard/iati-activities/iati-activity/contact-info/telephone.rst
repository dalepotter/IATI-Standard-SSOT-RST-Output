telephone
=========

``iati-activities/iati-activity/contact-info/telephone``

This is the reference page for the XML element ``telephone``. See also the relevant overview page: :doc:`/activity-standard/overview/contact-info` 

.. index::
  single: telephone

Definition
~~~~~~~~~~


The contact telephone number. May be repeated for
multiple numbers.


Rules
~~~~~

The text in this element must be of type xsd:string.








This element may occur any number of times.
