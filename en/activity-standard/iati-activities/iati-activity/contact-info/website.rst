website
=======

``iati-activities/iati-activity/contact-info/website``

This is the reference page for the XML element ``website``. See also the relevant overview page: :doc:`/activity-standard/overview/contact-info` 

.. index::
  single: website

Definition
~~~~~~~~~~


The contact web address. May be repeated for multiple sites.


Rules
~~~~~

The text in this element must be of type xsd:anyURI.








This element may occur any number of times.
