email
=====

``iati-activities/iati-activity/contact-info/email``

This is the reference page for the XML element ``email``. See also the relevant overview page: :doc:`/activity-standard/overview/contact-info` 

.. index::
  single: email

Definition
~~~~~~~~~~


The contact email address. May be repeated for multiple
addresses.


Rules
~~~~~

The text in this element must be of type xsd:string.








This element may occur any number of times.

~~~~~~~~~~~~~
